Welcome to Tweet Trends!

Tweet Trends is part of the Data Trends Django Based Web Application hosted by Heroku. The application uses a Python version of the Twitter API at the backend to extract live tweets and trending topics and perform Sentiment Analysis (SA) on the top trends. The results of the Sentiment Analysis is displayed using Pie Charts. Also, it allows a user to search for Hash-Tags and see the Sentiment Analysis on the tags. The sub-applications can be accessed by selecting the Tweet Trends List dropdown at the top left of this page.


App Contents
