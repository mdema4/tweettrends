 About!

Below are few questions that you may have regarding this web application.
Please e-mail me if you have further questions.



  • Q1. Why do you pick Twitter data?
    • According to Internet Live Stats, people post 6,000 tweets every second and about 500 million tweets per day. Working with this huge amount of LIVE data provides a great opportunity to test the performance of a Machine Learning Algorithms. Due to the limitation on the Twitter API, however, one can only query a certain number of tweets for a given twitter account. Even under this cap, one can get enough tweets to test a given algorithm.
      •

  • Q2. How many tweets per page request are used in this application?
    • Due to the cap on number of tweet query in a given time slot, only {{count_max_tweet}} tweets per a given trend are used to generate the Sentiment Analysis result.

  • Q3. What language is used in the application?
    • Only tweets and trends in English language is supported at the moment. Other languages are filtered before performing Sentiment Ananlysis.
      •

  • Q4. What modeling procedures have you followed?
    • Preprocessing: Mainly used regular expression (re) package to remove user identifiers and hashtags in the tweet and also implemented token "normalizer" that converts tokens to basic form by processing successive character (such as converting tokens like woow, wooooow and woooooow to woow).
    • Classification: To perform the sentiment analysis, I have implemented a Binary Naive-Bayes classifier and trained the model using 1.6 million distantly supervised training tweets and tested it using 500 manually labeled tweets. The accuracy of the model on the test data is 80%.
    • Note: Initially, I was planning to use the Scikit-Learn library but due to dependency of the package and lack of an easy-to-use buildpack in Heroku, I have decided to implement my own preprocessor, tokenizer and a Binary Naive Baye's Classifier.

  • Q5. Where can I find the training dataset?
    • The training dataset for the Sentiment Analysis can be downloaded from Sentiment140.
      •

  • Q6. What Python modules are used?
    • Mainly Regular Expressions, CSV, Tweepy, Matplotlib and Django framework.
